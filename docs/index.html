After all, the main things that can be seen are existed in the <header> ... </header> , <main> ... </main>  and <footer> ... </footer> tags. Header containes the dashes and the right panel, Main containes every thing inside the web-page and footer contains the final things inside the web-page. Mainly, footer and header would be the same in every page, but main will usually differ.

Customization

Below, there are some amazing tips on how would you CUSTOMIZE this theme, template or item.

Fonts Customizations

At /assets/css/fontsMgr.css, you would see that you have 6 blocks of code: 

@font-face
{
	•••
}
You need to change the value of the src: url(" ... ") not to damage anything. You will see that the font-family: ••• shouldn't be changed unless you know how to manage it.

Color Customization

At /assets/css/colors.css, you will also be able to change the colors of the entire website by editing what comes after the #. NOTE: never change the name if the color it self. For example, --blue:, but take a note of where does the blue works, and what color should replace it. Then change only the value, which comes after the : or #. You are able to put hex, rgb, hsl, cmyk or any other format of coloring.
